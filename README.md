# 웹 포트폴리오 사이트 제작

이 프로젝트는 "웹퍼블리싱 10년차 이상 경험을 어필하는 프론트엔드 구직을 위한 포트폴리오 사이트"를 제작하는 것입니다.

## 프로젝트 개요

### 무엇을 만드는가?

-   프론트엔드 개발자로서의 기술력과 경험을 효과적으로 어필할 수 있는 포트폴리오 웹사이트
-   다양한 프로젝트 경험을 DB에 저장하고 관리할 수 있는 시스템
-   방문자들에게 프로젝트 경험, 기술 스택, 연락처를 효과적으로 전달
-   관리자는 로그인해서 프로젝트 정보를 등록/수정할 수 있음

### 페이지 구성

1. **홈페이지** (`/`) - 자기소개와 주요 프로젝트 하이라이트
2. **프로젝트 목록** (`/projects`) - 모든 프로젝트 표시
3. **프로젝트 상세** (`/projects/{id}`) - 프로젝트 상세 정보
4. **연락처** (`/contact`) - 문의 양식
5. **로그인 페이지** (`/login`) - 관리자 인증
6. **관리자 페이지** (`/admin`) - 프로젝트 및 메시지 관리 (로그인 필요)

## 프로젝트 목표 및 디자인 방향

-   사용자 친화적이고 직관적인 인터페이스
-   모던하고 세련된 디자인으로 기술적 전문성 전달
-   모바일부터 데스크톱까지 완벽한 반응형 지원
-   부드러운 애니메이션과 인터랙션으로 사용자 경험 향상
-   빠른 로딩 속도와 최적화된 성능

## 프로젝트 구조

```
/
├── frontend/                  # Next.js 프론트엔드
│   ├── app/                   # App Router 기반 구조
│   │   ├── page.tsx           # 홈페이지
│   │   ├── projects/          # 프로젝트 관련 페이지
│   │   ├── contact/           # 연락처 페이지
│   │   ├── login/             # 관리자 로그인
│   │   └── admin/             # 관리자 페이지
│   ├── components/            # 재사용 가능한 컴포넌트
│   ├── lib/                   # 유틸리티 함수
│   └── public/                # 정적 파일 (이미지 등)
│
├── backend/                   # FastAPI 백엔드
│   ├── app/                   # 메인 애플리케이션
│   │   ├── api/               # API 엔드포인트
│   │   ├── db/                # 데이터베이스 연결 및 모델
│   │   ├── core/              # 핵심 설정 및 보안
│   │   └── utils/             # 유틸리티 함수
│   └── tests/                 # 백엔드 테스트
│
└── docs/                      # 프로젝트 문서
    ├── design_guide.md        # 디자인 가이드라인
    ├── api_schema.md          # API 스키마
    ├── db_schema.md           # 데이터베이스 스키마
    └── wireframes/            # 와이어프레임 이미지
```

## 기술 스택

### 백엔드

-   Python + FastAPI
-   MySQL 8.0 (utf8mb4 인코딩 필수)
-   aiomysql (SQLAlchemy ORM 사용 안함)
-   JWT 인증

### 프론트엔드

-   Next.js 15.x (App Router)
-   React 19.x
-   TypeScript
-   Tailwind CSS 4.x (CSS 기반 설정)

### 배포

-   Docker Compose로 3개 컨테이너 관리
    -   MySQL 데이터베이스
    -   FastAPI 백엔드
    -   Next.js 프론트엔드

## 개발 단계

1. **초기 설정 및 환경 구성**

    - 프로젝트 구조 및 기본 파일 생성
    - 개발 환경 설정 (Docker, Next.js, FastAPI)

2. **데이터베이스 및 API 개발**

    - 데이터베이스 스키마 구현
    - RESTful API 엔드포인트 개발
    - 인증 시스템 구현

3. **프론트엔드 UI 개발**

    - 컴포넌트 라이브러리 구축
    - 페이지 레이아웃 및 스타일 적용
    - 반응형 디자인 구현

4. **기능 통합 및 테스트**

    - 프론트엔드와 백엔드 통합
    - 사용자 인터페이스 테스트
    - 버그 수정 및 성능 최적화

5. **배포 준비 및 실행**
    - Docker 컨테이너 설정
    - 배포 스크립트 작성
    - 실제 환경에서 테스트
